# RankingSystem - Система ранжирования игроков в киберспортивной лиге
Система эффективно хранит рейтинги игроков, позволяет быстро обновлять их и получать список лучших игроков.


Программа построена на двух основных классах — Player и RankingSystem, а также пользовательском исключении PlayerNotFoundException. Архитектура опирается на принципы ООП: инкапсуляцию, чёткое разделение ответственности и использование встроенных коллекций с заданным порядком сортировки.
